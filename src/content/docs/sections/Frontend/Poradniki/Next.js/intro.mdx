Cześć! Witaj w handbooku Next.js. Znajdziesz w nim poradniki jak korzystać z Next'a w pełni i zacząć bawić się w Fullstack React Developera.

## Wstęp

Jako że jesteś członkiem Solvro, zakładam, że wiesz, że Next istnieje, ale korzystasz raczej tylko z file system routingu w App Routerze, use client i use server, bo debugger na ciebie nakrzyczał i hooków React'owych. Jednak Next.js posiada wiele więcej ciekawych możliwości i funkcji, poza używaniem use server wszędzie tam, gdzie robisz zapytania do API. To co, zaczynamy?
